# NotenManagerOSS
NotenManagerOSS is a Tool build for Libreoffice Calc or other Spreadsheet Programms. It is build for the German Grade System and can calculate your average Grade, your End Grade for every Subject and calculates a Grande Point average.   German Version: NotenManagerOSS berechnet die Durschschnittsnote und die Endnote pro Fach und den Zeugnisschnitt.
